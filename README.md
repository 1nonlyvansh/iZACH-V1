# iZACH — intent Zenith Adaptive Cognitive Handler

 A full-stack AI assistant with voice control, automation, and a custom Electron-based desktop UI.

Built as a modular system combining AI, automation, and real-time interaction.


## What Makes iZACH Different

- Not just a chatbot — a **system-level assistant**
- Multi-service architecture (backend + UI + automation agents)
- Voice → AI → Action pipeline
- Real integrations (Spotify, WhatsApp, system control)


## Core Features

- 🎤 Voice command processing (Speech → AI → Execution)
- 💬 Chat + command interface
- 📱 WhatsApp automation
- 🎵 Spotify control
- 🧠 Memory + context system
- ⚡ Task orchestration engine
- 🖥️ Custom Electron UI (React + Tailwind)


## Architecture

UI (React + Electron)
↓
Flask Backend (Python)
↓
Command Engine
↓
Modules (Spotify, WhatsApp, AI, etc.)
